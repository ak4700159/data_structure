
공백 = '\t'->탭, '\n'->개행(엔터), ' '->스페이스바

### 파일 스트림 지원 X (STDIN=키보드->프로그램, STDOUT=프로그램->모니터)
1. scanf, printf : 서식문자를 활용한 입출력 함수
	- scanf
		- %d, %f, %lf .. : 공백 무시(건너뜀)
		- %s : 공백전까지 읽음
			- %s 앞의 공백 무시함
		- %c : 공백 그 자체를 문자로 읽음
			- scanf(" %c", %ch) : 앞의 공백을 모두 무시하고 실제 문자를 읽음
			- 서식 문자열의 공백은 입력에서 하나 이상의 공백 문자와 대응됨
		- 입력 성공 시 입력된 변수의 수 만큼 반환함
	- 입출력 실패 시 scanf=0 또는 EOF, printf=음수 반환

2. gets, puts : 문자열 입출력 함수
	- gets : fets 함수 사용 권장(C11 표준에서 보안 문제로 인해 삭제된 함수)
	- puts : 출력할 문자열 + 개행문자와 함께 출력
	- 입출력 실패 시 gets=NULL, puts=EOF 반환

3. getchar, putchar : 문자 입출력 함수
	- getchar
	- putchar
	- 입출력 실패 시 EOF 반환

### 파일 스트림 지원 O 
1. fscanf, fprintf : 텍스트 기반 입출력 함수(서식문자 활용)
	- int fscanf(FILE *stream, const char *format, ...)
		- 지정한 파일에서 포맷에 맞춰 데이터를 읽어 변수 저장
		- scanf 서식문자 문법과 동일
		- 입력 성공 시 입력된 변수의 수 만큼 반환함

	- int fprintf(FILE *stream, const char *format, ...)
		- 지정한 파일에서 포맷에 맞춰 데이터 출력
		- printf 서식문자 문법과 동일
	- 입출력 실패 시 fscanf=0 또는 EOF, fprintf=음수 반환

2. fgets, fputs : 텍스트 기반 문자열 입출력 함수
	- char *fgets(char *str, int n, FILE *stream)
		- 입력 종료 기준
			- N - 1개 문자 읽었을 때
			- 개행 문자 읽었을 때
			- 파일 끝 EOF를 만났을 때
		- 항상 문자열 끝에 널문자를 붙이기 때문에 N - 1개 문자까지 읽음
		- 개행 문자까지 읽기 때문에 str[strcspn(str, "\n")] = '\0'; 많이 사용됨

	- int fputs(const char *str, FILE *stream)
		- 단순 출력 함수
	- 입출력 실패 시 fgets=NULL, fputs=EOF 반환

3. fgetc, fputc : 텍스트 기반 문자 입출력 함수
	- int fgetc(FILE *stream)
		- 파일 지시자가 가리키는 문자를 읽는 함수(사실 위 1번 2번 입출력 함수들도 파일 지사지를 기반으로 동작됨)
			- 파일 지시자를 통해 파일의 몇 번째 바이트를 읽고 있는지 관리 가능 
		- 문자를 읽은 후 파일 지시자는 한칸 뒤로 움직임
		- 파일 지시자와 관련된 함수로 feof, fseek, ftell 등이 있으며 이를 활용해 아래 기능 구현 가능
			- 파일 크기 구하기
			- 파일 다시 읽기
			- 특정 위치의 데이터 읽기
			- 특정 위치의 데이터 수정
			- 현재 위치 저장 후 나중에 복귀
			- 등...
	- int fputc(int char, File *stream)
		- 단순 문자 출력 함
	- 입출력 실패 시 EOF

4. fread, fwrite : 바이너리 기반 입출력 함수
	- size_t fread(void *ptr, size_t size, size_t count, FILE *stream)
		- 파일에서 크기가 size인 요소를 count 만큼 데이터를 읽어 ptr에 저장하는 함수
		- 실제로 읽은 요소 수를 반환
		- 파일 끝에 도달할 경우 count보다 적은 개수를 읽게됨

	- size_t fread(void *ptr, size_t size, size_t count, FILE *stream)
		- ptr에서부터 size 크기인 요소를 count만큼 파일에 저장하는 함수
		- 실제로 쓴 요소 수를 반환
